# BIOBUILDS Brand Guidelines

Use these guidelines when building projects for BIOBUILDS.

## Layout Padding

| Element Type | Padding |
|--------------|---------|
| Text / other items | 5% |
| Images / video | 3% |
| Section top/bottom | 0 |

## Color Palette

### Bright Mode
| Element | Color |
|---------|-------|
| Title | `#14171c` |
| Paragraph | `#737579` |

### Dark Mode
| Element | Color |
|---------|-------|
| Title | `#e6e8ea` |
| Paragraph | `#77787a` |
